# 扩展主图标下载指南

请从iconfont.cn下载以下SVG格式的图标，并保存为对应文件名：

1. `icon16.svg` - 16×16像素
2. `icon48.svg` - 48×48像素
3. `icon128.svg` - 128×128像素

在iconfont.cn上寻找适合作为扩展主图标的图标时，建议搜索"技术检测"、"AI助手"、"聊天机器人"等关键词。

## 下载步骤

1. 访问[阿里巴巴图标库](https://www.iconfont.cn/)
2. 找到合适的图标后，点击图标进入详情页
3. 点击"SVG下载"按钮
4. 将下载的SVG文件重命名为上述文件名，并放入此目录

## SVG图标优势

使用SVG格式有以下优点：
- 矢量图形，在任何尺寸下都保持清晰
- 文件大小通常比PNG小
- 支持Chrome扩展的自适应暗色/亮色模式
- 在高分辨率显示器上效果更好

## 创建不同尺寸的SVG

您可以使用同一个SVG文件，只需调整viewBox和宽高属性：

```svg
<svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24">
  <!-- 图标内容 -->
</svg>
```

如果您的SVG不包含尺寸属性，请在文件中添加width和height属性。
